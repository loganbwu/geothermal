5.8 Packages (and the people who made them)

This project was highly dependent on the JAGS developer Martyn Plummer, without whom I would have to use another package or code my own custom Metropolis-Hastings implementation - frightening stuff.

Also, being able to use the Rstudio and Rmarkdown developer environment made my job a whole lot easier, running code chunks and seeing the results inline. Additionally, this site was generated using Bookdown, straight from my own source code with some modifications. Bet you haven’t seen a compendium like this before!
